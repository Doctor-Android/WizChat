## 🚀 **How to Use It**

### **Prerequisites**
- C++17 compiler (GCC 7+, Clang 5+, or MSVC 2017+)
- CMake 3.16+
- Linux, Windows, or macOS

### **Installation**

```bash
# Clone the repository
git clone https://github.com/wiznet/wiznet.git
cd wiznet

# Build the system
mkdir build && cd build
cmake ..
make

# Run WizNet
./wiznet
```

### **What Happens When You Run It**

